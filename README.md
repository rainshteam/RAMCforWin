# RAMCforWin

Windows 高级内存清理工具 · 释放物理内存

---

## 是什么

一款 C++ 编写的 Windows 内存优化工具，集成 **7 项内存清理技术**，释放大量空闲内存。

---

## 环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 7 / 8 / 10 / 11 |
| 运行权限 | **管理员** |
| 编译工具 | Visual Studio 2013+ |
| 依赖库 | 无（仅系统原生 DLL） |

---

## 快速开始

```bash
RAMClean.exe      # 运行并显示结果
RAMClean.exe -q   # 安静模式（供其他程序调用）
```
